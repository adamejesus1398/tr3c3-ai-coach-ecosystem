# TR3C3 Digital Ecosystem

This is a comprehensive suite of personal productivity and growth tools designed for high performance.

## Applications Included

- **TR3C3 AI Coach**: An intelligent chatbot powered by AI to help you tackle your daily goals.
- **Home Dashboard**: Central hub for all applications.
- **Habits Tracker**: System for monitoring and maintaining daily habits.
- **Finance Dashboard**: Tool for managing personal finances and trading.
- **Digital Discipline**: Focus and productivity management.
- **Interior & Growth**: Tools for personal development.
- **Roadmap 1k Goal**: Tracking progress towards specific financial or personal milestones.
- **Routine Accordion**: Managing daily routines efficiently.
- **Daily Journal**: A reflective writing space.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```
2. **Open the application**:
   Simply open `index.html` in your favorite web browser.

## AI Coach Configuration

To use the AI Coach:
1. Open the AI Coach module.
2. Click the **Settings (⚙️)** icon.
3. Choose your provider (OpenRouter or Ollama Cloud).
4. Enter your API Key.
   - Note: For Ollama Cloud, the app uses a built-in CORS proxy to ensure functionality directly from the browser.

## Deployment

This app is designed to be hosted on **GitHub Pages**. Simply enable GitHub Pages in the repository settings and point it to the root directory.
